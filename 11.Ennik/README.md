# Calculate sum and difference
Funktsioonile antakse ette kaks täisarvu. Funktsioon peab tagastama nende kahe arvu summa ja nende kahe arvu vahe. Kui funktsioon tagastab mitu väärtust, siis see tegelikult ongi ennik. Näiteks:
return a, b, c
See tagastab enniku, ehk siis see on umbes sama kui:
return (a, b, c)

# Create a tuple from two numbers
Funktsiooni argumentideks on kaks täisarvu.
Kui arvud on võrdsed, tagastab funktsioon enniku ühe väärtusega (üks nendest sisendarvudest).
Kui arvud ei ole võrdsed, tagastab funktsioon enniku, kus esimesel positsioonil on esimese parameetri väärtus ja teisel positsioonil on teise parameetri väärtus.

# Merge tuples
Funktsioon saab sisendiks kaks ennikut.
Funktsioon liidab ennikud kokku, ehk teise enniku elemendid lisatakse esimese enniku lõppu.
Funktsioon tagastab enniku, kus on järjest esimese ja teise sisendiks saadud enniku elemendid.

# Create a tuple up to n
Funktsioon saab argumendiks ühe täisarvu n.
Funktsioon tagastab enniku järestikuste arvudega 0, 1, 2, .. n.
Kui n < 0, siis tagastab funktsioon tühja enniku.

# Remove odd numbers
Funktsiooni argumendiks on täisarvude ennik.
Funktsioon tagastab enniku, kus on argumendina saadud ennikust alles vaid paarisarvud (paaritud arvud "eemaldatakse").
Siin tuleb tähele panna, et ennikust elemente eemaldada ei saa. Küll aga saab luua uue enniku, kuhu lisame need elemendid, mida tahame tulemuses näha.
