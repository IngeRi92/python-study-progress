# Järjendi alustõed
Create a list of elements
Funktsiooni argumentideks on kaks täisarvu.
Funktsioon tagastab listi, kus on need kaks elementi samas järjekorras nagu nad funktsiooni sisse antakse.
Create new list with added number
Funktsiooni sisenditeks on järjend täisarvudest ja üks täisarv.
Tuleb tagastada uus järjend, mis koosneb algsest järjendist ja sellele lisatud arvust.
Algset järjendit ei tohi muuta.
Created repeated list
Funktsiooni sisenditeks on järjend ja täisarv.
Funktsioon tagastab uue listi, mille elemendid on esialgse järjendi elemendid sisendina saadud arv kordi.
Reverse a list
Funktsiooni sisendiks on järjend.
Funktsioon tagastab järjendi, kus esialgse järjendi elementide järjekord on ümber pööratud.
Vihje: kasuta lõiku (slice)

# Create a list of edge elements
Funktsiooni sisendiks on järjend täisarvudest.
Funktsioon tagastab kahe-elemendilise järjendi, kus esimesel kohal on algse järjendi esimene element ja teisel kohal algse järjendi viimane element.
Algses järjendis on vähemalt üks element. Ühe elemendi puhul tuleb seda sama elementi käsitleda kui esimest ja viimast elementi.

# Create new list from edge elements
Funktsiooni sisendiks on vähemalt kahe elemendi pikkune järjend.
Funktsioon tagastab järjendi, mis on algne järjend ilma esimese ja viimase elemendita.

# Add element in position
Funktsiooni sisenditeks on järjend, uus element ja uue elemendi positsioon listis.
Positsioon on alati korrektne, ei saa tekkida olukorda, kus elemendi lisamine on võimatu.
Uus element tuleb lisada järjendisse ette antud positsioonile.
Kõik elemendid alates sellest positsioonist (kaasa arvatud) nihutatakse ühe võrra paremale.
Positsioonide lugemine algab nullist, nagu indekseerimise korral.
Positsioon võib olla ka järjendi lõpus - sellisel juhul lisatakse uus element lõppu.

# Get the middle element of a list
Funktsiooni sisendiks on järjend, mis koosneb paaritust arvust elementidest.
Funktsioon tagastab järjendi keskmise elemendi väärtuse.
