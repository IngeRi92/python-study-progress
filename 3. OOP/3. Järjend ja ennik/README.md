# Leia suurimad elemendid

Antud on täisarvude järjend, tagasta järjend, mis sisaldab ainult esialgse järjendi maksimaalseid elemente. Kui sisendis on üksik element, mis on kõige suurema väärtusega, tuleb tagastada list, kus on üks element sees. Kui sisendis on näiteks kõige suurema väärtusega elemente kolm, siis tuleb tagastada järjend kolme elemendiga.

# Loe kokku paaris- ja paaritud arvud

Loe, kui palju paaris ja kui palju paarituid arve leidub antud numbrite järjendis. Funktsioon tagastab tulemuse enniku kujul (odds, evens)

# Eralda nimed sisendist

Kirjuta funktsioon, mis saab tekstina inimeste skoorid ning tagastab listi nimedest, kes on piisavalt suure skoori saanud.

Sisendis on tulemused eraldatud komaga (,). Iga tulemus koosneb nimest ja täisarvulisest skoorist. Skoor on mitte-negatiivne täisarv. Nimi võib koosneda mitmest nimeosast (eraldatud ühe tühikuga). Nimeosad võivad koosneda tähtedest, numbritest ja sümbolitest (välja arvatud koma). Funktsioon peab tagastama need nimed, kus skoor on suurem-võrdne kui etteantud min_result.

# Filtreeri ja muuda järjendit

Funktsioon saab sisendiks ennikudest koosneva järjendi ja täisarvu. Funktsiooni väljundiks on filtreeritud järjend, mis koonseb ennikudest, mille summa on väiksem või võrdne täisarvuga.
