# Tehted
Ülesandes tuleb realiseerida järgmised tehted:
Tuleb liita a ja b ning välja printida tulemus.
Tuleb lahutada c-st d ning printida vahe.
Tuleb leida e ja f-i korrutis ning see välja printida.
Tuleb leida g ja h jagatis ning see välja printida.
Tuleb leida i jääk j-ist ning see välja printida.
Tuleb leida k täisarvuline jagatis l-st ning see välja printida
Tuleb leida m astmes n väärtus ning see välja printida.
Tuleb leida kas o on suurem või võrdne p-st ning prindi True
Tuleb leida kas r on väiksem või võrdne q-ga ning prindi välja True
Tuleb leida kas s on võrdne z-ga ning prindi välja True väärtused on võrdsed
Tuleb leida kas t ei ole võrdne u-ga ning prindi välja True väärtused ei ole võrdsed
Tuleb leida ristahuka ruumala etteantud height, width ja height väärtustega ning see välja printida.
Tuleb ajaühikud teisendada minutiteks. Antud on days, hours, minutes, seconds. Vaata näiteid mallist.

# Ülesanne: ATM
Tuleb koostada raha väljamakse automaat, millega saab välja võtta kindlaid rahatähti. Sisendiks on summa, mis on vahemikus 1 - 500 ning prindib välja, mitu erinevat rahatähte on vaja, et sisestatud summa katta. Eesmärk on katta summa võimalikult väheste rahatähtedega. Masinas on järgmised rahatähed: 1€, 5€, 10€, 20€, 50€, 100€.
Näide:
On antud summa 56€. Vaja läheb selle summa tagastamiseks kolme rahatähte: 50€, 5€, 1€.
