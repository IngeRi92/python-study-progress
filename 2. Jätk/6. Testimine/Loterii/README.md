# Loterii

Teie programm peab oskama otsustada loterii võidu suurust. See loterii mäng on hästi lihtne ning võit sõltub vaid kolmest väärtusest: a, b ja c.

Kui kõik kolm arvu on "võidu numbrid" ehk 5, siis tagastatakse suurim võidu tulemus - 10

Kui kõik kolm arvu on lihtsalt samad, aga ei võrdu "võidu numbriga", siis tagastatakse keskmine võidu tulemus 5

Kui nii b, kui ka c on erinevad arvust a, siis võidu tulemuseks on 1

Muul juhul, kui kas b või c on võrdne a väärtusega, võidu tulemuseks on 0

a - int: arv a

b - int: arv b

c - int: arv c
