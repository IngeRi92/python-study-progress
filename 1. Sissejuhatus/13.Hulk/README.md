# Hulga alustõed
Create a set from numbers
Funktsiooni argumentideks on kolm täisarvu ning funktsioon tagastab hulga neist arvudest.
Convert list to set
Funktsiooni argumendiks on järjend, funktsioon teeb sellest hulga ja tagastab selle.
Count unique elements in the list
Funktsiooni argumendiks on järjend ning see tagastab, mitu unikaalset (mitte-korduvat) elementi selles järjendis on.
Remove sixes
Funktsiooni argumendiks on hulk numbritega.
Funktsioon eemaldab hulgast kõik kuued ja tagastab hulga ilma kuuteta.

# Find numbers divisible by 3
Funktsiooni argumendiks on järjend täisarvudest.
Kõik täisarvud on vahemikus 0 kuni 1000 (kaasa arvatud).
Funktsioon tagastab hulgana sisendjärjendist need arvud, mis jaguvad kolmega.

# Add one to numbers in the set
Funktsiooni sisendiks on hulk täisarvudega.
Funktsioon suurendab iga hulgas olevat arvu ühe võrra ja tagastab hulga neist ühe võrra suurendatud arvudest.

# Get uninvited friends count
Funktsiooni argumentideks on järjend sõprade nimedest ja järjend kutsutud külaliste nimedest.
Funktsioon tagastab, mitu sõpra jäi kutsumata.
Kutsutute seas võib olla ka inimesi, kes ei ole sõbrad.

# Contains duplicates
Funktsiooni argumendiks on järjend.
Funktsioon tagastab, kas antud järjendis esineb duplikaate (korduseid).
Kui järjendis on vähemalt kaks samaväärset elementi, tagastab funktsioon True, muul juhul False.
