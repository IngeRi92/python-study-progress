# Remove vowels from a string

Funktsioon saab sisendiks sõne original_string, tagasta see sõne ilma täishäälikuteta (aeiou).
Kui sõne koosneb ainult täishäälikutest, peaks väljund olema tühi sõne.
Täishäälikute kustutamise asemel on mõistlik teha uus sõne, kuhu täishäälikuid ei lisata.

# Calculate compound interest

Raha summas amount eurot paigutatakse pangas hoiusele years aastaks, mis teenib aastas intressi rate protsenti. Tänu liitintressile hakkab raha kasvama.
Kasutades tsüklit tagasta lõppsumma, mis pärast years aastat hoiusele on kogunenud.
Liitintressi üldine valem on järgnev: result = amount _ (1 + rate) \*\* years, või result = amount _ (1 + rate) ühe aasta kohta.
