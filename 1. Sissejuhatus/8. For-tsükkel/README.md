# Contains digit

Funktsiooni sisendiks on sõne. Ülesande eesmärk on leida for-tsükli abil, kas sõnes on number või mitte. Kui sõne sisaldab numbrit tagastatakse True, kui sõnes ei ole ühtegi numbrit, siis False.
Meetod .isdigit() aitab kontrollida, kas sõnes olev sümbol on number.

# Find digit sum

Funktsiooni sisendiks on sõne kujul täisarv numbers, tagasta selle ristsumma.

# Sum of numbers in a range

Funktsiooni sisendiks on kaks täisarvu start ja end.
Tagasta arvude summa start ja end vahel, mõlemad arvud on kaasa arvatud.
Võid arvestada, et start on alati väiksem kui end või võrdne.

# Sum of even numbers in the range

Funktsiooni sisendiks on täisarv n, n >= 0.
Tagasta kõikide paarisarvude summa kuni ette antud arvuni (kaasa arvatud).

# Make hola string

Funktsioon saab sisendiks arvu, mis näitab, mitu korda peab sõna "hola" sisalduma funktsiooni väljundis.
Kuigi sõne kordamiseks saab seda korrutada täisarvuga ("hola" \* count), siis kasuta siin for-tsüklit, et tulemus kokku panna.
