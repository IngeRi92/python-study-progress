# Get hobby
Funktsiooni sisendiks on sõnastik hobbies ja sõne name.
Sõnastikus on võtmed inimeste nimed ja väärtused selle inimese hobi, nt: {"tom": "running"}.
Tagastada ette antud inimese (name) hobi.
Sisend ei ole tõstutundlik.

# Count symbol appearances
Funktsiooni sisendiks on sõne.
Tagastab sõnastiku, kus võti on sümbol ja väärtus on arv, mitu korda see sümbol sõnes esineb.

# Find people taller than the height limit
Funktsiooni sisendiks on sõnastik, kus võtmed on inimeste nimed ja väärtused on nende pikkused, ja
täisarv, millest pikem peab inimene olema.
Funktsioon tagastab listi nimedest, kes on pikemad kui etteantud limiit.

# Swap keys and values
Kirjutada funktsioon, mis saab sisendiks sõnastiku, vahetab selle võtmed ja väärtused üksteise vastu ära ning tagastab selle.
Kõik väärtused on unikaalsed - olukorda, kus uued võtmed ühtivad, ei teki.

# Remove sixes
Funktsiooni sisendiks on sõnastik, kus väärtused on täisarvud.
Tagastab sõnastiku, kus kõik kuuega jaguvate väärtustega võtmepaarid on eemaldatud.
