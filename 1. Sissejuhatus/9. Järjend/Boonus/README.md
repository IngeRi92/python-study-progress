# Sum between 2 and 5

Kirjuta funktsioon, mis ette antud järjendist summeerib kokku arvud, mis jäävad 2 ja 5 vahele (2 ja 5 ei ole kaasa arvatud). Ehk siis 2-st järgmisest arvust alustatakse liitmist ja enne 5 lõpetatakse ära.

# Swap edge elements

Funktsiooni sisendiks on vähemalt kahe elemendi pikkune järjend.
Funktsioon tagastab järjendi, kus algse järjendi esimene ja viimane element on vahetatud.

# Odd sums of consecutive elements

Sisendiks täisarvude järjend. Järjendi iga kõrvutiolevat elemendi summat vaadeldakse. Kui see summa on paaritu arv, siis lisatakse see summa tagastatavasse järjendisse. Ehk siis tulemuseks on järjend, milles on kõik paarituarvulised kahe järjestikuse elemendi summad.
