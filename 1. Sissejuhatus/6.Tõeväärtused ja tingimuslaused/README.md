# Are equal
Funktsiooni sisendiks on kaks arvu num_a ja num_b.
Tagastab "equal" kui arvud on võrdsed ja "not equal" kui ei ole.
Võrdsust saab kontrollida topelt võrdusmärgiga (==).

# Positive, negative or zero
Funktsiooni sisendiks on täisarv.
Funktsioon tagastab "negative" kui antud arv on negatiivne, "positive" kui positiivne ja "zero" kui antud arv on null.

# Is letter in a word
Funktsiooni sisendiks on täht ja sõna.
Funktsioon peab tegema kindlaks, kas antud täht on sõnas või ei, ning vastavalt selle tagastama True või False.

# Are strings the same length
Funktsioon saab sisendiks kaks sõne.
Tagastada True kui sõned on sama pikad, ja False kui ei ole.
Sõne pikkuse saab kätte kasutades len() funktsiooni, nt. len("car") => 3.

# Is letter or digit
Funktsioon saab sisendiks ühe sümboli ning peab tagastama kas tegemist on tähe, numbri või muuga.
Tasub kasutada meetodeid isalpha() ja isdigit().
Ülesande lahendamiseks leiad abi: Sõne sisseehitatud meetodid PyDoc.
